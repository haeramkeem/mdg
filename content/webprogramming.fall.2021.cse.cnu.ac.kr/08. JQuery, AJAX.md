## CDN Installation

```html
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
```

## Basic

### Document onload

```jsx
$(document).ready(function() {
// JQueries
});

$(function() {
// JQueries
});
```

### Selector

```jsx
$(css-selector).JQueryMethod();
```
